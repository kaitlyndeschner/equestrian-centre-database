# KD Equestrian Centre Database

## Overview

This project is a SQL database system designed for managing information related to an equestrian centre.

The database stores and manages data such as:

* Customers
* Horses
* Staff
* Lessons
* Bookings
* Payments
* Events

The project demonstrates relational database design, table creation, data insertion, and SQL querying.

## How to Use

### Requirements

You will need a database management system that supports SQL, such as:

* MySQL
* SQL Server
* MariaDB
* PostgreSQL

### Running the Project

1. Download or clone the repository.
2. Open your SQL database management software.
3. Open the file:

```text
kd-equestrian-centre.sql
```

4. Execute the SQL script.

The database, tables, and sample data will then be created automatically.
